## Objective
To help citizens easily report civic issues using voice or text without knowing government procedures.

## Functional Requirements
- User can type complaint
- User can speak complaint (voice)
- System classifies issue using NLP
- System detects department automatically
- System generates complaint text
- User can upload image
- Complaint history tracking

## Non Functional Requirements
- Works in Telugu/Hindi/English
- Simple UI for rural users
- Fast response
- Mobile friendly
